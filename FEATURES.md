# SnapCuller Feature Guide 🚀

Comprehensive documentation of SnapCuller features designed for an efficient and professional culling workflow.

---

## 🛠 Sidebar: Management & Configuration

### 1. Project Management
*   **Change Folder**: Quick directory access via click or drag-and-drop. Includes a **Progress Bar** to track culling completion and percentages.
*   **Workflow Preset**: Save complete application states—including presets, templates, and folder locations—to reuse for recurring event types or projects.

### 2. Metadata & Automation
*   **Code Replacements**: Use `=code=` macros for instant metadata entry. Supports bulk data import via `.tsv` files from Excel or other spreadsheet software.
*   **IPTC Metadata Template**: Create reusable metadata profiles (Author, Copyright, Description, etc.) to ensure data consistency across your catalog.
*   **Batch Rename**: Rename multiple files at once to standardize your asset naming conventions in a single step.

### 3. Selection Tools
*   **Tactical Buckets**: The core of a keyboard-driven workflow. Map specific keys (e.g., `1`) to instantly move or copy selected formats (like RAW only) to target physical folders.
*   **Advanced Filter**: Smart filtering based on Ratings, Color Labels, or specific EXIF data such as ISO, Lens, and Camera model.

### 4. Info & System
*   **EXIF & IPTC Editor**: Technical capture info and a non-destructive metadata editor. All changes generate `.xmp` sidecar files compatible with Lightroom, Capture One, and Photoshop.
*   **Customization**: Full keyboard shortcut remapping, Gamepad controller support, and visual theme customization.

---

## 🖼 Navbar: Visual Tools & Navigation

### 1. Workspace Layout
*   **Single View**: Full focus on a single image for detailed inspection.
*   **Grid View**: Gallery layout for quick navigation and bulk asset management.
*   **Compare View**: Side-by-side comparison for up to 4 images to help you pick the best frame among similar shots.

### 2. Technical Assist
*   **Triple-State Exposure**: Visual technical check. 1st click (Highlights), 2nd click (Shadows), 3rd click (Both).
*   **Focus Peaking**: Visual highlights on the sharpest areas to verify focus without manual zooming.
*   **AI Face Zoom**: Automatic face detection with instant zoom to verify expressions and sharpness on human subjects.
*   **Image Orientation**: Instant rotation tool to fix portrait images that appear in landscape orientation.

### 3. Smart Organization
*   **RAW+JPG Pairing**: Merges RAW and JPG files into a single visual entity to reduce workspace clutter.
*   **Burst Stacking**: Automatically groups high-speed burst shots into a single stack for cleaner browsing.

---

## 📦 Grid View: Batch Operations

When in Grid Mode, you have access to powerful bulk management controls:

*   **Bulk Selection**: Quickly select large sequences of photos for status changes or moving to buckets.
*   **Batch IPTC Edit**: Apply metadata templates to all selected photos simultaneously.
*   **Professional Export**:
    *   **Destination Folder**: Define target paths for selected assets.
    *   **Subfolder Grouping**: Automatically organize exported files into specific sub-folders (e.g., 'Approved').
    *   **Selective Format**: Option to export only specific formats (e.g., Keep only RAW for selected shots).
    *   **Export Modes**: Choose between *Copy* (duplicate files) or *Move* (physical file transfer).
*   **Smart Grouping**: Visual grouping by date or metadata to simplify management of long shooting sessions.
